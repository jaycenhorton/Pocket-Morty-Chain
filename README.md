# Pocket-Morty-Chain
Adopt your own Pocket Morty and register that ownership on the Ethereum blockchain!

## Setup
### Pre-reqs
- NPM, node, Metamask chrome extension
1. `npm install -g ethereumjs-testrpc`
1. `npm install -g truffle`
1. `testrpc`
1. `truffle compile`
1. `truffle migrate`
1. `npm run dev`
## Test contracts
- `truffle test`
